Operation

The alert button should generate an alert when pressed, as a 'button' input.
The text input produces an event on every change to the input text.
The first pair of radio buttons must have at least one set; a log message is produced on the callback when any one is selected
The second pair of radio buttons need not have any set; a log message is produced on the callback when any one is selected
The two checkboxes are independent; each produces a (different) log message indicating whether they are selected on each click
The first dropdown is a *single* with a requirement for at least one being selected; a callback is invoked when any is chosen.
The second dropdown is a *multiple* entry with no requirement for any to be set; a callback is invoked (with the *latest* value selected) when any is chosen - all of the selected values are available in event.target.selectedOptions.
The first input range is a *float* with value 100 to 110
The second input range is an *integer* with value -50 to 50
The third input range has a step of 10 with value -500 to 0
